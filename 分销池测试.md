主要测试商城总分销奖池的分发以及分支分销奖池的分发功能。

1.商城总分销奖池的分发的前提条件

- [x] 存在配置好的门槛
- [x] 存在满足分享条件的用户
- [ ] ~~用户之间存在分销关系~~
- [x] 存在完成的订单

2.分支分销奖池的分发前提条件

- [x] 存在配置好的门槛
- [x] 存在满足分享条件的用户
- [ ] 用户之间存在上下级关系
- [ ] 存在已经完成的订单

1.注册三个普通用户（记录id）

2.第一个用户邀请三个新用户，并记录新用户id

3.第二个用户邀请三个新用户，其中一个新用户还有一个子用户

4.第三个新用户只有一个子用户

5.把门槛设置为1，1，1，1

6.让所有新用户买一个测试员工支付产品
